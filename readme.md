

# Installation
- Clone the repository
```
git clone https://github.com/Philippe-Collignon/rest-api-node-jwt-typescript.git
```
- Install dependencies
```
cd to <repo>
npm install
npm run build
```
- Launch server & client:
```
docker-compose build
docker-compose up
```
